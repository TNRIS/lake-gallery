Lake Story Map - Mobile Instructions

Important: The lake story map requires the use of two fingers to pan and zoom around the lake area on a mobile device.

The map has restricted bounds based on each lakes' boundary. The map also has restricted minimum zoom (how far you can zoom out).

To view the available tools in a lake story view, click the tools menu icon in the top right of the map.

In the tools menu, some lakes have links to historical imagery collections via the TNRIS DataHub. To view an available DataHub collection for a lake story:

  1. Open the tools menu in the top right of the map by clicking/touching the menu icon.
  2. Click/touch a year on the swiper tool to add the overlay historical layer to the map.
  3. Click/touch the same year again to open a new browser tab for that historical layer on the DataHub.

For any other questions or concerns, please read through the Lakes of Texas About page and use the Contact form if needed.
